# Définition
La suite de Fibonacci est définie par $$\begin{cases} X_0 = 0 \\ X_1 = 1 \\ \forall n \in \mathbb{N},\ X_{n+2} = X_{n+1}\ + X_{n} \equiv \forall n \in \mathbb{N},\ X_n = X_{n-1} + X_{n-2} \end{cases}$$
