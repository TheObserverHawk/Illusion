# Définition
La fonction tuple prennent les arguments de différents types et de les retourner en tuples 

#### Exemple 
```python
s = 'abc' 
t = range(3) 
d = {'a':12, 'b':17}

tuple(s)  # ('a', 'b', 'c') 
tuple(t)  # (0, 1, 2) 
tuple(d)  # ('a', 'b')
```
