# Définition
La fonction `list` permet de convertir un objet en liste, si c’est possible. 
Cela fonctionne notamment avec les objets de type `range` et de type `str`.
#### Exemple
```PYTHON
	list(range(4))    # [0, 1, 2, 3]
	list(range(2, 6)) # [2, 3, 4, 5]
	s = 'hello'
	list(s)           # [’h’, ’e’, ’l’, ’l’, ’o’]
```
