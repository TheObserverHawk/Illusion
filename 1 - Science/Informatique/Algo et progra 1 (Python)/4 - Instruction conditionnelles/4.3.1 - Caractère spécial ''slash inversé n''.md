# Définition
Le caractère spécial `\n` peut être utilisé dans un `str` et représente une nouvelle ligne

#### Exemple 
```python
name = input('Quel est votre nom ?\n') 
Quel est votre nom ? 
John Doe

print(name)  # John Doe
```
