# Définition
Soit $f$ : $A$ ⟼ $B$ une fonction
Le domaine de $f$ : Dom$f$ = { a ∈ $A$ | ∃ b ∈ $B$, $f$(a) = b }

_Remarque_ :
1. A l'ensemble des éléments de départ
2. B l'ensemble des éléments d'arrivé
3. $f$ une fonction
