# Définition
Lors du dé-référencement d'une méthode, il est nécessaire d'ajouter les valeurs de ses arguments. 
## Syntaxe
La syntaxe devient : 
```java
referenceInstance . nomSymbole ( valeursArguments )
```

#### Exemple
```java
Carre carreBleu = new Carre(); 
carreBleu.deplacer(5, 7);
```
