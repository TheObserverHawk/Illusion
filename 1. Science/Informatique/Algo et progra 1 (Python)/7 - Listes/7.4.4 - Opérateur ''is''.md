# Définition
L'opérateur `is` permet de tester si deux variables réfèrent le même objet.

#### Exemple 
```python
a = 'banana'
b = 'banana'
a is b  # True


a = [1, 2, 3] 
b = [1, 2, 3]
a is b  # False
```
