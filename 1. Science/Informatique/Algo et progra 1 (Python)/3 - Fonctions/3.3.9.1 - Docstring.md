# Définition
Un docstring est un commentaire multilignes placés au début du corps d’une fonction 
#### Exemple 
```python
def get_sum(x, y):
""" return the sum of x and y """
	return x + y

help(get_sum) 
# Help on function get_sum in module __main__:
# get_sum(x, y)
#	 return the sum of x and y
```
